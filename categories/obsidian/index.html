Obsidian Plugin "Typst PDF Export" v1.2.0 released!

Sep 10, 2025

New release of my new Obsidian plugin Typst PDF Export:

Version 1.2.0 comes with a few bug fixes and a lot of refactoring.

Typst PDF Export uses Pandoc and Typst to export professionally typset PDFs from your notes.

It supports the basic Markdown syntax as well as a few special code blocks, like an e-mail block. Also it not only puts your note’s images in the PDF, it also creates a preview page of PDFs linked in your note. And if you want your exported PDF to be extra special, you can embed any non-image file type into the PDF as attachment.
